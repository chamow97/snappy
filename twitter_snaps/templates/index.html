Snappy - Help

General Instructions

  • Snappy is a tool to display images in Twitter/Tubmlr with given tags.
  • User should have an account to use this application.
  • Previous results can be shown by checking the feature.
  • Requests are limited. App might not work after some huge searches in short span of time. Use it after some time

Twitter Hashtag Search

  • Search the hashtag.
  • The User and the Tweet itself of the searched tweet can be viewed by hovering over the image.
  • Click an image to open the gallery.

Tumblr Tag Search

  • Search the tag.
  • The User and the content of the results can be viewed by hovering over the image.
  • Click an image to open the gallery.
